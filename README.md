# Opencv-Object-Tracking

## Introduce
Detect object **without** using YOLO. The tracker I use **SORT** method (Kalman Filter + Hungarian Algorithm).
## Requirement
1. python3.8
2. scikit-image
3. matplotlib
4. numpy
5. opencv
6. filterpy
## Run
```markdown
python datect_with_track.py
```
